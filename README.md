# VST.DotNet-STMData
## Getting started
```cmd
git clone
cd VST.STMData.ClientExample
dotnet restore
dotnet build
```
## Sample usage
### Config
```json
  "AccessToken": "<YourAccessToken>"
```
### Run
```cmd
cd dotnet-fcdata
dotnet restore
dotnet build
dotnet run --project vst-dotnet-stmdata/VST.STMData.ClientExample.csproj
```
